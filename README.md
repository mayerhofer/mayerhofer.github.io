# mayerhofer.github.io - A dashboard with some of my private projects

Here I have a portal website that leads to information about me and about projects done by me.

## Prerequisites

- Node.js (Latest LTS version recommended)
- npm (Comes with Node.js)

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/mayerhofer/mayerhofer.github.io.git
cd mayerhofer.github.io
npm install
```

## Development

To run the application in development mode with hot reload:

```bash
npm start
```

The application will be available at `http://localhost:1234`

## Building for Production

To create a production build:

```bash
npm run build
```

The built files will be available in the `public` directory.
